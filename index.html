About
Siapa Kami?

EBOOKR VVIP adalah komunitas independen para peneliti keamanan, ethical hacker, dan bug bounty hunter dari seluruh Indonesia. Didirikan oleh sekumpulan passionate security enthusiast yang percaya bahwa kolaborasi adalah kunci dalam dunia cybersecurity.

Kami aktif berpartisipasi di platform bug bounty global seperti HackerOne, Bugcrowd, dan Intigriti, dengan fokus pada web application security, API vulnerabilities, Android application penetration testing, dan network penetration testing.

Bergabunglah bersama kami untuk berbagi pengetahuan, writeup, dan pengalaman di dunia offensive security.
